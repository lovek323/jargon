\mainentry{bits} pl.n.

\begin{inparaenum}
    \item Information. Examples, ``I need some bits about file formats.' '(``I
        need to know about file formats.'') Compare \citesense{core dump}{4}.
    \item Machine-readable representation of a document, specifically as
        contrasted with paper: ``I have only a photocopy of the Jargon Fil,e;
        does anyone know where I can get the bits?'' See \citeentry{softcopy},
        \citeentry{source of all good bits}. See also \citeentry{bit}.
\end{inparaenum}
