\mainentry{blast}

\begin{inparaenum}
\item v.,n. Synonym for \citeentry{BLT}, used esp. for large data sends over
    a network or comm line. Opposite of \citeentry{snarf}. Usage: uncommon.
    The variant `blat' has been reported.
\item vt. [HP/Apollo] Synonymous with \citesense{nuke}{3}. Sometimes the
    message ``Unable to kill all processes. Blast them (y/n)?'' would appear
    in the command window upon logout.
\end{inparaenum}
