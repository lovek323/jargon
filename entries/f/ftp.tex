\mainentry{FTP} /F-T-P/, not /fit'ip/

\begin{inparaenum}
	\item [techspeak] n. The File Transfer Protocol for transmitting files between systems on the Internet.
	\item vt. To \citeentry{beam} a file using the File Transfer Protocol.
	\item Sometimes used as a generic even for file transfers not using \citeentry{FTP}.
		\inlinequote{Lemme get a copy of \worktitle{Wuthering Heights}\ ftp'd from uunet.}
\end{inparaenum}
